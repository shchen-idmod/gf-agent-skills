# finance skills

Skills in this directory are scoped to the finance team.

## Adding a skill

1. Create a folder: `skills/<your-skill-name>/`
2. Copy the [skill template](../../../docs/SKILL_TEMPLATE.md) into it as `SKILL.md`
3. Set `tier: group` and `group: finance` in the frontmatter
4. Run `python scripts/validate_skill.py` locally
5. Open a PR — group skills require 1 approval from your team lead
