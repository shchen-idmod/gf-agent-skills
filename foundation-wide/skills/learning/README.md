# learning

Skills in this category are available to all GF staff.

## Adding a skill

1. Create a folder: `learning/<your-skill-name>/`
2. Copy the [skill template](../../../../docs/SKILL_TEMPLATE.md) into it as `SKILL.md`
3. Run `python scripts/validate_skill.py` locally
4. Open a PR — foundation-wide skills require 2 approvals from the Skills Working Group
